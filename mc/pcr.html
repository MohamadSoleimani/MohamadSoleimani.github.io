Multi-Channel Real-Time PCR System

Project Overview

This project marks the first effort in Iran to develop a multi-channel real-time PCR system. My master's thesis, "Optimizing the Temperature Changes of the Heating Cycle in Real-Time PCR," focuses on improving the device's heating and cooling systems. The PCR method involves a thermal cycle that operates between 65 to 95 degrees Celsius. Below is a picture of the device.

Figure 1: Multi-Channel Real-Time PCR Device

Image Processing and Thermal Cycle Control

To control the device's thermal cycle, we utilized an Arduino Uno board connected to a Raspberry Pi. The images captured by the Raspberry Pi were processed using the OpenCV library in Python. Below is an example of a processed image.

Figure 2: Processed Image using OpenCV

Real-Time Monitoring of PCR Amplification Using Fluorescent Detection

As the DNA molecules replicate, they emit fluorescent light, thanks to the inclusion of fluorescent dyes in the reaction mixture. This fluorescence signals successful DNA amplification. We use a Raspberry Pi camera to detect and measure the intensity of this light, enabling real-time monitoring of the PCR process.
